Парсер вакансий по навыкам на hh.ru

Информация о парсере


Парсер осуществляет поиск по сайту hh.ru навыков и знаний которыми кандидату необходимо обладать при желании получить ту или иную работу. Например, при поиске по ключевой фразе разработчик python парсер выдаст общее количество вакансий в которых встречается это сочетание и список навыков, которые заявлены работодателем в вакансии.

Поиск всегда осуществляется по ключевому слову или фразе. Навыки выводятся при каждом поиске.

В фильтрах есть дополнительные параметры поиска. Это регион и сферы деятельности, по которым сгруппированы все вакансии на hh.ru. При выборе искать 'везде' - поиск будет осуществляться по всем сферам деятельности.

Парсер отсортирует все навыки, которые были найдены по ключевому слову во всех вакансиях по нисходящей от более востребованных к менее востребованным.
